¿Cómo leer este reporte?
Déficit El modelo predice que las ventas de este producto bajarán más de lo normal. Considere reducir su pedido o buscar un precio de oferta para agotar existencias.
Superávit El modelo predice que las ventas subirán más de lo habitual. Asegúrese de tener suficiente inventario para no quedarse sin existencias.
Estable Las ventas se esperan dentro del rango habitual. Mantenga su pedido estándar.
★ Alerta Prioritaria El producto presenta una desviación significativa respecto a su promedio de ventas mensual. Son los productos que más impacto tendrán en sus ganancias si no se toma acción.
